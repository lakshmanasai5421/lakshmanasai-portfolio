Who I Am

Data, ML & cloud — end to end

I'm a Python Developer & ML Engineer focused on data science, machine learning, and cloud-native deployment. I love building systems that take raw data all the way to scalable, production-ready models.

On the cloud side, I work with Microsoft Azure — using Azure ML for training pipelines, AKS for Kubernetes orchestration, and Blob Storage for data management. I containerise applications with Docker, manage deployments with Kubernetes, and automate infrastructure using Terraform.

My ML experience spans NLP with BERT and Transformers, time-series forecasting with LSTM & ARIMA, and classical classification — all connected to real-world backends and cloud services.
